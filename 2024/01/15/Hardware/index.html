常用元器件 ( 分立元件 )

1.1 电阻

  • 电阻分为直插电阻, 贴片电阻, 热敏电阻, 电位器(变阻器)等
  • 一般来说只考虑贴片电阻
  • 常用的封装有0603和0805
  • 封装数字越大他的大小越大
  • 封装和功率大小是相关的, 但还未明白这个的实际意义
  • 现在的理解是封装只涉及大小, 然后值是额外填 的.

1.2 电容

常用作用 :

  • 储能, 滤波

一般用的多的电容 :

  • 贴片电容MLCC, 瓷片电容, 电解电容.

特性 :

  • 直流电流无法通过电容, 电阻电压为0

储能作用 :

滤波作用 :

电容的主要参数 :

  • 容量, 耐压

直插点解电容:

  • 常用在给电源升压模块后的电压滤波, 用直插电解电容的一般是大电压, 然后电容也是比较大的, 用来过滤低频的波, 小的电容一般用来过滤高频波

1.3 电感

性质 : 电流惯性

  • 其实这个东西和电容很想, 有电压和电流惯性

作用 :

  • 和电容类似 : 也是储能和滤波

重要参数 :

  • 电感量 , 饱和电流

和电容区别 :

  • 未知

1.4 二极管和LED

常用二极管 :

  • 普通二极管, 稳压二极管, TVS二极管, (肖特基二极管, 快恢复二极管)(高频二极管), 发光二极管

二极管性质 :

  • 单向导通

二极管参数 :

  • 最大电流, 正向压降, 反向压降

稳压二极管 :

  • 正常的二极管方向击穿后就坏了, 但是稳压二极管专门工作在反向击穿去反向击穿后, 电压会维持在一个特定值

实际应用 :

  • 阻容降压

整流 :

作用 :

  • 整流作用

TVS二极管 : 瞬态电压抑制二极管

  • 可以保护电器设备不受导线引入的电压尖峰破坏, 可以对尖峰电压削顶

1.5 三极管和光耦

三极管的作用:

  • 用作电子开关, 信号的放大

控制原理:

  • 有三个电极 : e(发射极) b(基极) c(集电极)
  • 小电流控制大电流, 小电流就像是水龙头的止水阀一样, 调节基极, 基极的电流越大, 集电极的电流就会有越多被放出

NPN三极管 :

PNP三极管 :

  • 实际上PNP就像是NPN三极管的对称, 日常工作中使用得更多的是NPN三极管

三极管的日常应用 :

  • 流控型开关, 有点就能导通

  • 这里会有一个重要的应用, 三极管可以拿来做高低电平使能
