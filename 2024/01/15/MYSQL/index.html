1-5 SQL分类

根据操作和目标分为三类 :

  1. DDL (Data Definnition Language)(数据定义语言) 操作对象主要是表和库 如: 建表, 建库
  2. DML (Data Manipulation Language)(数据查询语言) 操作对象主要是数据记录 如: 对表中的查询操作
  3. DCL (Data Con Language)(数据控制语言) 操作对象主要是数据记录 如: 对用户的权限的设置

1-4 SQL通用语法

  1. SQL语句可以单行或多行书写, 以分号结尾
  2. 可使用空格和缩进来增强语句的可读性
  3. MySQL 数据库的 SQL 语句不区分大小写, 关键字建议大写书写
  4. 3种注释
  • 单行注释 : – (右边需要一个空格) 或 #(不用空格)
  • 多行注释 : /* * / 和 c语言一样
