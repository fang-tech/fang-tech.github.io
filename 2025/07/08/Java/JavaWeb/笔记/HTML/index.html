表单标签

  • 用于向特定服务器提交内容
  • 两种method的区别 :
    • get : 数据会到action中填写的url后面url?name1=value&name2=value…
    • post : 数据会通过请求体发送, 不会后缀在url后

表单项标签

单选框

  • name => 用于将单选框分组, 组内被选是互斥的, value => 提交上去的表单中sex的值

复选框

下拉框

按钮

隐藏域

用于设置固定会提交的内容, 不需要也不希望用户看到的表单内容, 比如用户的uuid等

多行文本框

  • 没有value选项, 可以通过在双标签中间填入内容来设定默认内容

span => 用作对层的划分
